Image Optimization with Cloudinary

2days ago by
Jonathan Graef

We always looking for ways to improve site speed. We lazy load, minify, and concatenate, create sprites, experiment with caching, and just about everything else we can think of.

The idea behind image optimization is simple but execution hasn't always been:each of those optimizations usually requires a separate solution which makes automation difficult. Enter Cloudinary:uploading, storing, amnipulating, and delivering images and videos on the web.

Using Cloudinary

There are many use cases for a service like Clodinary. WhenI think about my site, I see one ovious need: resized,optimized images on post list pages. Google PageSpeed crucifies my sitefor image size and I an't blame them. I have a regex rouitine that searches for the first image in a post and uses that as the post image, but rouitine doesn't otimize images, so if the image is very large, I may scale it down with HTML attributes but the download size is still massive.

1,2,3,4,5,6

If you are looking to serve media via the <\picture> element for different media queries, Cloudinary has an excellent structure for doing that as well.

Hosting your videos with Cloudinary will allow you to have the same advantages of manipulation and performant cloud hosting as your images will have!

Try Cloudinary!

Cloudinary is an awesome one-stop shop for uploading, storing, manipulating, and delivering images and video. The APIs are all intuitive, effective, and easy to use. The optimization plugs are all usefull and the included face detection, sepia/greyscale transformations, and social media image creation a huge, handy bonus.
